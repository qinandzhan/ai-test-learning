# 🚀 AI 自动化测试用例生成器 (Docker 版)

这是一个基于面向对象架构封装的 AI 测试工具，调用阿里云通义千问大模型，只需输入功能名称，即可秒级生成企业级异常测试用例。

## ✨ 核心特性
* **面向对象封装**：代码结构优雅，易于扩展接入其他大模型。
* **安全隔离**：核心 API Key 采用环境变量动态传入，拒绝代码硬编码，保障资产安全。
* **容器化交付**：提供完整 Dockerfile，无视系统环境差异，开箱即用。
